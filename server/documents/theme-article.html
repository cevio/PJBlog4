内页数据源pageCustomParams.article

id 日志ID
title 日志标题
category 日志分类
id 分类ID
name 分类名
info 分类说明
icon 分类图标
url 分类连接地址
content 日志内容
postDate 日志发表日期
editDate 日志修改日期
tags 日志标签信息
id 标签ID
name 标签名
url 标签连接地址
count 标签数量
user 日志编写者信息
photo 用户头像
name 用户昵称
poster 是否为poster用户权限 true or false
oauth 登入方式
login 是否已经登入
logindate 最后登入日期
loginip 最后登入IP
views 日志查看次数
other 日志其他相关信息
other.prev 上一篇日志信息
id 日志ID
title 日志标题
url 日志连接地址
cover 日志封面
other.next 下一篇日志信息
id 日志ID
title 日志标题
url 日志连接地址
cover 日志封面

    <div class="information">
        <div class="category">Category: <a href="<%=pageCustomParams.article.category.url%>" title="<%=pageCustomParams.article.category.info%>"><%=pageCustomParams.article.category.name%></a></div>

                for ( var tagitems = 0 ; tagitems < pageCustomParams.article.tags.length ; tagitems++ ){
        %>
            <a href="<%=pageCustomParams.article.tags[tagitems].url%>"><%=pageCustomParams.article.tags[tagitems].name%></a>
        <%			
                }
        %>
        </div>
        <%	
            }
        %>
    </div>
</div>
